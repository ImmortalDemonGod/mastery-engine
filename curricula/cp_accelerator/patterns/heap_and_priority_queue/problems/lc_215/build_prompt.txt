# Build Challenge: Kth Largest Element in an Array

## Pattern Overview

> Heaps are specialized tree-based data structures that satisfy the heap property, making them efficient for finding minimum or maximum elements. Priority queues are abstract data types typically implemented with heaps.

## Problem Statement

Given an integer array `nums` and an integer `k`, return the `kth` largest element in the array.
Note that it is the `kth` largest element in the sorted order, not the `kth` distinct element.
Can you solve it without sorting?
